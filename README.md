Program sederhana untuk melakukan pengambilan data dari social media (twitter) menggunakan API dan dikirimkan ke telegram (telegram API)
CPSSSIMBOT

- OAuth 1.0 & OAuth 2.0

- 
- 

### requirements
- Windows 10, Ubuntu 18.04 x64
- PostgreSQL v12
- Nodejs v14
- npm v6
- Twitter API
- Telegram API

### install
```telegram API
npm i slimbot
```
```Twitter API
npm install twitter
```
